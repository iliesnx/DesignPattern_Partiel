# ShopEase - Système de Notifications

## Design Patterns (Version Simple)

**Observer** : Un objet notifie automatiquement d'autres objets quand quelque chose se passe

**Strategy** : Différentes façons de faire la même chose (envoyer email, SMS, etc.)

**Factory** : Usine qui crée des objets sans savoir comment ils sont faits

**Command** : Emballe une action dans un objet pour l'exécuter plus tard

## Fichiers par Pattern

- observer.js : Exemple du pattern Observer (notification automatique)
- strategy.js : Exemple du pattern Strategy (canaux interchangeables)
- factory.js : Exemple du pattern Factory (création d'objets)
- command.js : Exemple du pattern Command (actions encapsulées)
- notification_system.js : Système complet ShopEase (tous les patterns ensemble)
